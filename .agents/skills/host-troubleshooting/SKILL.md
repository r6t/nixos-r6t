# Host Troubleshooting

For goldenball freezes or related hardware symptoms, read `docs/GOLDENBALL_FREEZES.md` before proposing changes.

Distinguish recorded observations from hypotheses. Preserve known-good mitigations and do not generalize goldenball-specific findings to mountainball or other hosts without evidence.

Do not run rebuild, activation, or destructive diagnostic commands. After flake edits, run `./format.fish`.
